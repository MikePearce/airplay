<task type="auto">
  <name>Task 1: Create the MQTT-to-ALSA volume bridge script</name>
  <files>bridge.py</files>
  <action>
Create bridge.py in the project root — a single Python script that:

1. Imports: paho.mqtt.client (with CallbackAPIVersion.VERSION2 for paho-mqtt 2.x), subprocess, logging, sys

2. Configures logging to stdout with format: "%(asctime)s %(levelname)s %(message)s"

3. Defines MQTT config:
   - BROKER = "localhost"
   - PORT = 1883
   - TOPIC = "shairport-sync/remote"

4. Defines command mapping dict:
   ```python
   COMMANDS = {
       "volumeup": ["amixer", "sset", "PCM", "5%+"],
       "volumedown": ["amixer", "sset", "PCM", "5%-"],
       "mutetoggle": ["amixer", "sset", "PCM", "toggle"],
   }
   ```

5. Defines on_connect callback that:
   - Logs "Connected to MQTT broker, subscribing to {TOPIC}"
   - Subscribes to TOPIC
   - Uses reason_code (v2 API) to check connection success

6. Defines on_message callback that:
   - Decodes payload to string, strips whitespace
   - If payload is in COMMANDS dict: runs subprocess.run(cmd, capture_output=True, text=True, timeout=5)
   - Logs the command and result (stdout)
   - If payload is "playpause", "nextitem", or "previtem": logs "Ignoring {payload} (not supported)" at DEBUG level
   - If payload is anything else: logs "Unknown command: {payload}" at WARNING level
   - Wraps subprocess.run in try/except (subprocess.SubprocessError, OSError, subprocess.TimeoutExpired) — logs error and continues, never crashes

7. Defines on_disconnect callback that logs disconnection

8. Main block:
   - Creates mqtt.Client(callback_api_version=CallbackAPIVersion.VERSION2)
   - Sets on_connect, on_message, on_disconnect
   - Calls client.connect(BROKER, PORT)
   - Calls client.loop_forever()
   - Wraps in try/except KeyboardInterrupt for clean shutdown with client.disconnect()

Make the script executable (add shebang: #!/usr/bin/env python3).

Do NOT add argparse, config files, or environment variable reading — that is Phase 4 scope (RELY-02). Keep it simple: hardcoded config, single file, ~60 lines.
